有关此博客的一些说明

¶经过两天的努力，终于算是搞定了初版博客，有几点需要说明：

¶建站

此博客基于hexo+github建立，网上教程很多，简易方法可参照：https://www.cnblogs.com/fengxiongZz/p/7707219.html

建议可以多看看hexo官网https://hexo.io/zh-cn/ 的文档

Markdown编辑器个人采用Typora，感觉挺好用

¶功能

  1. 设置 Archives和 Tags 两种分类方式，分别支持以时间节点为筛选和以tags为筛选查看博客内容；
  2. 设置评论功能，评论引用Valine，支持表情，用户名及邮箱可不填，欢迎一起交流；
  3. Github和 Weibo可链接到本人Github和微博；
  4. Link中目前有旭升dalao的博客，之后会慢慢加一些有用的链接；
  5. 今后有空、有想发的会尽量发出来，希望大家多多关注，也希望能对大家有所帮助，希望大家多提建议，努力改进。
  6. 主题链接：https://github.com/yscoder/hexo-theme-indigo
