各种忙碌的一年

​ 从这个博客建立到现在已经过去一年多的时间，当初一时兴起建立，现在再看看也只是发了当初建立时的一篇，果然自己只是个受点兴趣驱使的人，而且兴趣过的很快，之后一切就被抛之脑后（狗头）。但说到底还是这一年多的确是忙如一匹憨憨，从准备考研->搞定保研->蜜汁毕设，再到现在算是正式入职，算半个职场人士吧。忙和累充斥着整个过程，现在终于有时间把这个之前扔下的东西重新捡起来了。

​ 重新开始写总要回顾一些之前的事情，重新回到羊姐身边是去年或者是这四年最重要的事，这件事对我影响很大，以至于我在很多次提起，感谢自己也感谢羊姐。

2018.07-08 之前都在所谓的复习考研，这段时间才是真正的复习考研。每天在和张宇、老蒋、肖秀荣斗争，哦对，当然少不了化原的三巨头。在408值班的日子是充实的，但是浪的属性依旧没变，一起值班的同学属实比我这个渣渣认真地多，但也在按自己的节奏复习，也没很清楚要怎么走，但想着一直在复习着总没错。

2018.09 是的，又开学了，但是本科的最后一个学年，没有特别的感受。同学们都在为傻X化院大四开始搞毕设的安排而骂娘，我也在整日画着图纸中开始了保研之路。准备各种材料，和老大聊，准备面试，最后好像只是走完了所有流程，回过神来也就已经搞定了所有事，不能确保这条路的前方是我所想的，但至少眼下的一切我觉得很OK。

2018.10 搞定一切赶到兰州参加了姐的婚礼，但兰州这鬼地方着实不想再去第二次，之后是又一次的燕化实习，还是熟悉的学习两小时，休息大半天，不同的是心情很轻松，虽然回来之后就开始了苦逼的坐班生活。
哦对，10月还有一件很重要的事，羊姐说她恢复的差不多了，接下来一起嗨皮！这是值得我特别开心的事，至少我的努力是有用的

2018.11-2019.02 一段四个月如一日的生活，每日日常坐班，而我又是那种坐不住的人，但每天在学新的东西，实验组的、用户体验设计的、软件开发的，需要接触很多东西就逼迫你不得不多去学一些内容，这可能算是一种反动力吧。1月底终于回到了家，计划了一大堆回家果然过起了日常的躺尸生活，但是难得的惬意的假期。
羊姐考研考的特别好，棒~

2019.03-05 魔鬼一样的毕设，每天都充斥着紧张和令人厌烦的气息，搞不出来的模拟，做不完的设计，写不完的论文。从开题到中期再到最终答辩，几近崩溃的情绪出现了好几次，可能是日常的各种杂事堆积在一起的确是感觉异常疲惫。自我努力调节并和羊姐聊过一两次之后自己的状态的确还算不错，一直都在成长吧，这两年或许是成长最快的两年。
羊姐面试特别顺利，果然分数高就是好，北邮她来了。

2019.06 毕业这个词很神奇，既充满了离别和伤感，又不乏喜悦。毕业季，校园里随处可见的学士服和拖着行李箱离开的人，渐渐解决完所有需要处理的事，大家来去匆匆，想要趁着这毕业的关口留下些许回忆，我也只是和兔子、旭升即将离京的伙伴告别，516的酒桌上小芳痛哭流涕，的确再见已不知何时。没有照片，送别各位朋友，转头和在身边的伙伴继续向前，人生总是要不得不告别。

2019.07 暑期学校，三年后又一次参与，只是角色不同，漫长的准备期，到最后是一节一节的课程。

新的路已经开始走了，还好你们都在，继续一起向前吧……
